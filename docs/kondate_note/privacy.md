# プライバシーポリシー（こんだてノート）
最終更新日：2026-05-23

本アプリ「こんだてノート」（以下「本アプリ」）は、以下の方針で個人情報等を取り扱います。

## 1. 収集する情報

### 端末内に保存される情報
- 家族人数・買い物曜日・NG食材などの設定情報
- 生成した献立・買い物リストのデータ

これらの情報は端末内にのみ保存され、当方のサーバーへは送信されません。

### 外部サービスへ送信される情報

**Google Gemini API（献立生成）**  
献立を生成する際に、以下の情報をGoogle Gemini APIへ送信します。
- 家族人数
- NG食材リスト
- 直近で使用した料理名（重複回避のため）

氏名・メールアドレス・位置情報など、個人を特定できる情報は送信しません。  
参考：<https://policies.google.com/privacy?hl=ja>

**RevenueCat（サブスクリプション管理）**  
プレミアムプランのご利用にあたり、RevenueCatを使用します。RevenueCatは購入・サブスクリプションの管理目的でデバイス識別子等を取得する場合があります。  
参考：<https://www.revenuecat.com/privacy/>

**Apple（App内課金）**  
サブスクリプションの決済はAppleのApp内課金を通じて行われます。支払い情報はAppleが管理し、当方は取得しません。  
参考：<https://www.apple.com/jp/legal/privacy/>

## 2. 利用目的
- AI（Gemini）を利用した献立・買い物リストの自動生成
- プレミアムプランのサブスクリプション管理

## 3. データの保管・削除
- アプリ内データ（設定・献立履歴）は端末内に保存されます。アプリを削除すると端末内のデータも削除されます。
- 外部サービス（RevenueCat）が保持するデータについては、各サービスのプライバシーポリシーに従います。

## 4. 第三者提供
上記の外部サービス連携（Google Gemini API・RevenueCat）を除き、利用者情報を第三者に提供することはありません（法令に基づく場合を除く）。
